## O que faço

Construo sistemas que transformam **territórios em dispositivos dialógicos** — onde memória, corpo, linguagem e dados se encontram para produzir conhecimento coletivo.

O meu trabalho cruza:

- **Curadoria de patrimônio cultural intangível** — cartografia de memórias orais, folclore, práticas comunitárias
- **Dramaturgia urbana** — teatro de invasão, intervenção no espaço público, corpo como arquivo
- **Sistemas digitais abertos** — plataformas de governança de dados, georreferenciação, participação cidadã
- **Metodologias dialógicas** — pedagogia lúdica, agir comunicativo, alteridade como método

## Projetos em curso

### 🗺️ Atlas Vivo MILK
Plataforma integrada de cartografia de memória intangível nos territórios portugueses.  
Georreferenciação de personagens folclóricos, práticas culturais e memórias vivas por freguesia.  
`Google Apps Script` · `Leaflet` · `Google Sheets` · `Drive API` · `EUPL`

### 👁️ OlHaPIN
Sistema de observação participativa de inteligência territorial.  
Sensores ambientais + dados comunitários + devolutiva às comunidades como ferramenta de negociação.  
`Smart City` · `Ambient Intelligence` · `Open Data` · `GDPR`

## Fundamentos metodológicos

| Origem | Conceito central | Aplicação |
|--------|-----------------|-----------|
| Dissertação de Mestrado (2009) | Dialogia lúdica (Bakhtin) | Múltiplas vozes coexistindo sem que nenhuma anule as outras |
| Teatro de rua / Invasão urbana (Carreira) | Território como dramaturgia | O espaço público é sempre uma estrutura dramatúrgica |
| Desenvolvimento simbólico (Piaget) | Jogo como forma de conhecer | O lúdico é modo epistémico de estar no mundo |

> *"O território é uma sala de aula que ainda não sabe que é um palco. O meu método é o acontecimento que os faz descobrir isso ao mesmo tempo."*
