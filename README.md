## O que faço

Construo sistemas que transformam **territórios em dispositivos dialógicos** — onde memória, corpo, linguagem e dados se encontram para produzir conhecimento coletivo.

- **Curadoria de patrimônio cultural intangível** — memórias orais, folclore, práticas comunitárias
- **Dramaturgia urbana** — teatro de invasão, intervenção no espaço público, corpo como arquivo
- **Sistemas digitais abertos** — governança de dados, georreferenciação, participação cidadã
- **Metodologias dialógicas** — pedagogia lúdica, agir comunicativo, alteridade como método
